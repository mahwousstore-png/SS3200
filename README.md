# ✨ مولّد أوصاف العطور | Perfume Description Generator

تطبيق ذكي لتوليد أوصاف HTML احترافية لمنتجات العطور بمكونات حقيقية — متوافق مع منصة سلّة.

## المميزات
- ✅ مكونات عطرية حقيقية من Fragrantica
- ✅ تنسيق HTML متوافق مع سلّة (بدون فراغات)
- ✅ تطبيق عام — اسم المتجر والرابط اختياري
- ✅ دعم التستر وعطور الشعر
- ✅ اختبار اتصال API مدمج
- ✅ عدة نماذج AI (مجاني + مدفوع)

## النشر على Streamlit Cloud
1. ارفع الملفات على GitHub
2. اذهب إلى [share.streamlit.io](https://share.streamlit.io)
3. اختر المستودع → `app.py` → Deploy

## الحصول على مفتاح API
- سجّل مجاناً في [openrouter.ai](https://openrouter.ai)
- أنشئ مفتاح API
- النموذج الافتراضي (Gemini Flash) مجاني
