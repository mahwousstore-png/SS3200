## طريقة النشر على Streamlit Cloud

### 1. رفع الملفات على GitHub
ارفع هذه الملفات إلى مستودع جديد على GitHub:
```
legabriel-app/
├── app.py
├── requirements.txt
├── .streamlit/
│   └── config.toml
└── README.md
```

### 2. النشر على Streamlit Cloud
1. اذهب إلى [share.streamlit.io](https://share.streamlit.io)
2. سجّل دخول بحساب GitHub
3. اختر المستودع → `app.py`
4. اضغط **Deploy**

### 3. الاستخدام
1. أدخل مفتاح **OpenRouter API** في الشريط الجانبي
2. ارفع ملف **غبريال تحديث.xlsx**
3. اضغط **توليد الأوصاف**
4. حمّل الملف المحدّث

## الحصول على مفتاح API
- سجّل في [openrouter.ai](https://openrouter.ai)
- أنشئ مفتاح API مجاني
- النموذج الافتراضي (Gemini Flash) مجاني
